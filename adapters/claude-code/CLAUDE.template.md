# Project Guidance

Use Amalgam Conductor as the router when ownership is unclear.

- Route UI/UX and accessibility to Cloak Meister.
- Route documentation to Scribe Meister.
- Route diagrams to Meister Weaver.
- Route databases to Meister Chronicler.
- Route normal QA to Acme Overseer.
- Route normal security/privacy review to Cipher Meister.
- Route destructive, negative, fuzz, failure-mode, or resilience testing to Hidden Dagger only after explicit selection and a completed safety gate.

Base findings on evidence, mark assumptions and missing evidence, and avoid project-specific claims that files do not support.
