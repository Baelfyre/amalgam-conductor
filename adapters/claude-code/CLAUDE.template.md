# Project Guidance

Use Conductor as the router when ownership is unclear.

- Route UI/UX and accessibility to Cloak.
- Route documentation to Scribe.
- Route diagrams to Weaver.
- Route databases to Chronicler.
- Route normal QA to Overseer.
- Route normal security/privacy review to Cipher.
- Route destructive, negative, fuzz, failure-mode, or resilience testing to Dagger only after explicit selection and a completed safety gate.

Base findings on evidence, mark assumptions and missing evidence, and avoid project-specific claims that files do not support.
