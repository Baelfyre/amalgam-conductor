# Local AI Adapter

Load skill Markdown into prompt context or a retrieval index. Smaller local models should load Amalgam Conductor's summary first, then one selected specialist `SKILL.md`, then only required supporting files. This repository does not claim that every local AI runtime can discover or execute skills automatically.

See [install-guide.md](install-guide.md), [local-system-prompt.template.md](local-system-prompt.template.md), and [retrieval-layout-guide.md](retrieval-layout-guide.md).
