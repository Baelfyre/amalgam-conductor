# Agent Instructions

Use Conductor when a task needs routing. Otherwise use the smallest relevant specialist:

- Cloak for UI/UX and accessibility.
- Scribe for documentation.
- Weaver for diagrams.
- Chronicler for databases.
- Overseer for normal QA and readiness.
- Cipher for normal security/privacy review.
- Dagger only after explicit selection and a completed non-production safety gate.

Verify facts from project files. Disclose missing evidence. Ask before destructive, production, publication, or Git history actions.
